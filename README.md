# LTC Block Chain



**ltc-blockchain** 是一个用于查询莱特币(LTC)区块、交易、地址等信息的系统.

## 介绍

 - ltc-blockchain使用 **Node.js** 开发，底层需要 **litecoind** 支持，基础功能已完成.
 - the project is under heavy developement.
